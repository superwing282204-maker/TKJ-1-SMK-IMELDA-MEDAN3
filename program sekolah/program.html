Rincian Materi

// arahkan kursor pada tiap kartu

NODE 01 · HARDWARE

Perakitan Komputer

Mengenal komponen fisik dan menyusunnya menjadi satu unit yang menyala dan berjalan normal.

  • Komponen motherboard, CPU, RAM, storage
  • Prosedur perakitan & pemasangan yang aman
  • Pengecekan POST dan BIOS/UEFI
  • Troubleshooting perangkat keras dasar
NODE 02 · SOFTWARE

Sistem Operasi

Memasang dan mengatur "otak" yang menjalankan seluruh perangkat keras yang sudah dirakit.

  • Instalasi Windows & distro Linux
  • Partisi disk dan manajemen boot
  • Driver dan konfigurasi dasar sistem
  • Perintah dasar command line
NODE 03 · JARINGAN

Jaringan Dasar

Menghubungkan satu komputer ke komputer lain hingga membentuk sebuah jaringan yang saling bicara.

  • Topologi jaringan (star, bus, mesh, ring)
  • Pengkabelan UTP & konektor RJ-45
  • Pengalamatan IP dan subnetting dasar
  • Perangkat: switch, router, access point
NODE 04 · LOGIKA

Pemrograman Dasar

Melatih cara berpikir sistematis sebelum menulis instruksi untuk mesin.

  • Algoritma dan flowchart
  • Struktur kontrol (percabangan, perulangan)
  • Dasar HTML sebagai pengantar
  • Logika pemecahan masalah
NODE 05 · TEORI

Sistem Komputer

Memahami cara kerja komputer dari dalam: bagaimana data diproses secara digital.

  • Konversi bilangan biner, oktal, heksadesimal
  • Arsitektur dan cara kerja CPU
  • Konsep input–proses–output
NODE 06 · DIGITAL

Simulasi & Komunikasi Digital

Menggunakan perangkat digital untuk berkomunikasi, berkolaborasi, dan mempresentasikan hasil kerja.

  • Kolaborasi dokumen daring
  • Simulasi visual & presentasi digital
  • Etika berkomunikasi di dunia maya
